# PyTorch Video Classification

## Getting Started

```
pip install -r requirements.txt
```

## Running the example

Download the data:

```
wget https://www.crcv.ucf.edu/data/YouTube_DataSet_Annotated.zip
```

Unzip it:

```
unzip YouTube_DataSet_Annotated.zip
```

Run the example:

```
python main.py
```
